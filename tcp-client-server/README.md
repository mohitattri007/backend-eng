# TCP Echo Service

A simple TCP client–server project demonstrating how TCP works at the
transport layer (OSI Layer 4).

## What this project demonstrates
- TCP connection establishment
- Reliable, ordered communication
- Client–server model
- OS-managed TCP sockets

├── server/
├── client/
└── README.md
